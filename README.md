# Multi-Disease Health Risk Prediction System
### Final Year Project — BSc (Hons) Data Science | February 2026 Intake

## Project Overview
An end-to-end machine learning system that predicts risk of 
Diabetes, Heart Disease, and Hypertension using CDC health data.
Includes SHAP explainability and a Streamlit web application.

## Tech Stack
Python | pandas | scikit-learn | XGBoost | SHAP | Streamlit
